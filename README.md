# GiftHunting - A Minecraft minigame

得分 > 80% 时，获得【决胜】Buff，始终赋予玩家发光效果，免疫所有负面效果

当场上有玩家持有【决胜】时，所有其它玩家解锁【猎杀黎明之弓】，每60秒获得一支【猎杀黎明之箭】

当持有【决胜】的玩家被【猎杀黎明之箭】击中时，掉落 10% 分数

【猎杀黎明之箭】可以被识破，被识破后，持有【决胜】的玩家获得【誓约】Buff，期间免疫【猎杀黎明之箭】
